# Operations

  - `+` addition (ints are special)
  - `-` subtraction (ints are special)
  - `*` multiplication (ints are special)
  - `/` division (always produces a float)
  - `//` integer division
  - `**` exponent
  - `%` modulo  
  - `<<` and `>>` bit shifts
  - `~` logical not
  - `&` and, `|` or, `^` xor
  
 Remember PEMDAS: parenthesis, exponents, mult, div, add, sub
 
 Use parenthesis!
